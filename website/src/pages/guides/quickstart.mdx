# Quickstart

Application 루트에 NaverMaps Provider를 제공해야합니다. 사용하시는 프레임워크에 맞춰 `<NavermapsProvider />` 를 감싸주세요.

``` jsx
import { NavermapsProvider } from 'react-naver-maps';

function App() {
  return (
    <NavermapsProvider 
      ncpKeyId='MY_NAVERMAPS_CLIENT_ID'
    >
      <TheRestOfYourApplication />
    </NavermapsProvider>
  )
}
```
